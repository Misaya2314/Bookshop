## 数据库设计调整

### books 集合字段调整

- `categoryId` -> `collegeId` (学院 ID)
- `subCategoryId` -> `majorId` (专业 ID，使用专业名称作为 ID)
- 新增 `collegeName` (学院名称)
- 新增 `majorName` (专业名称)

### 新增 colleges 集合

```javascript
{
  _id: "学院ID",
  name: "学院名称",
  icon: "学院图标",
  order: 排序号,
  status: "active/inactive",
  majors: [
    {
      id: "专业ID",
      name: "专业名称",
      order: 排序号
    }
  ]
}
```
